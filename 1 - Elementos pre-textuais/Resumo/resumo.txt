O ensino de lógica de programação para pessoas que nunca codificaram é um desafio presente desde o início da computação. No curso de Sistemas de Informação do IFMA-MC não é diferente, a metodologia de ensino baseada em codificação Portugol ainda se mostra com dificuldade para superar esse desafio e assim garantir um processo de ensino eficiente para alunos que nunca codificaram sejam capazes de avançar no processo de ensino de programação. Justamente neste ponto entra este trabalho e tem como objetivo demonstrar o uso de programação visual para contornar a dificuldade de codificar pela primeira vez, onde os alunos ingressantes poderão aprender primeiramente lógica e somente depois partir para a codificação. Para isso irei apresentar uma forma de ensino baseada em fluxograma e usarei o Mapler Interpretador Portugol, um ambiente de desenvolvimento projetado justamente para o ensino de lógica de programação por meio da pseudolinguagem Portugol e que apresenta entre as suas funcionalidades o desenvolvimento de fluxogramas para resolução de algoritmos.


Palavras-chave: Ensino. Lógica. Codificação. Metodologia. Fluxogramas.
