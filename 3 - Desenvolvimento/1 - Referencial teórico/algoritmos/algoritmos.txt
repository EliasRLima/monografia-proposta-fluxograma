Um algoritmo é uma sequência de passos finitos com instruções bem definidas, ao final destes passos é esperado que o objetivo seja alcançado. Durante o andamento do algoritmo o mesmo pode mediar diversas informações de entrada e saída, durante esse gerenciamento de informações a sequência de passos pode se tornar mais complexa conforme mais dados devam ser considerados.
Quando passamos para a área de computação, algoritmos e o processo de programação são quase acoplados e desta forma é fundamental que estudantes de programação entendam tambem o que é e como resolver algoritmos. Durante toda a historia se desenvolveu diversas formas de treinar raciocinio e resolução de algoritmos, foram usados desafios, charadas, calculos matemático, mapas mentais e diversas outras formas.
Na computação surgiram linguagens e psudolinguegem que tinham como objetivo facilitar o ensino de programação, para isso buscavam diminuir sua complexidade e ficar o mais próximo possível da linguagem nativa do estudante. No Brasil desde a década de 1970 é utilizado o Portugol, tambem conhecido como Português estruturado, e este é originalmente uma tradução da linguagem de programação Pascal para a língua portuguesa.
Com a popularização do Portugol surgiram diversos derivados deste e alguns até adaptados para outras linguagens de programação com C e Java. Esta pseudolinguagem é bastante utilizada para ensino de lógica e tem como vantagem que o estudante não necessite conhecer uma linguagem estrangeira podendo programar usando sua lingua nativa. O Portugol evoluiu muito desde sua primeira versão e já apresenta suporte para comandos, operadores, funções, recursividade e diversas estruturas de dados. 
Para utilizar o Portugol não necessariamente se faz necessário uma IDE, mas caso deseje rodar o código em Portugol como se o mesmo fosse de fato uma linguegem existem diversas ferramentas capazes de auxiliar com isso, como por exemplo o VisualG, o Portugol Studio e, a ferramenta que este trabalho irá utilizar, o Mapler - Interpretador Portugol.
